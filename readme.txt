== Installation ==

This section describes how to install the plugin and get it working.

1. You can either use the automatic plugin installer or your FTP program to upload plugin files to your wp-content/plugins/notifyvisitors directory. Make sure the source files reside in this directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your NotifyVisitors options (*Options - NotifyVisitors*) for Wordpress 2.3.x, (*Settings - NotifyVisitors*) for Wordpress 2.x.x-4.x.x 
4. Paste the key you received from https://www.NotifyVisitors.com/ or get it from your www.NotifyVisitors.com dashboard.
5. Get Secret Key and brandid (used in edit module setting) as explained in the following steps
<ul>
<li>Go to www.notifyvisitors.com and register</li>
<li>Go to the Integration section in sidebar</li>
<li>Note the BrandID and Secret Key</li>
</ul>
